  <h1>Python3.6+Flask+Mysql搭建的一个简单BBS.</h1>
前台页面Flask，使用sqlalchemy完成orm数据库表和类别的映射。数据库使用Mysql 5.7.22版本。</br>
实现个人用户注册、登陆、留言，主题和类别新增。</br>
要求：</br>
1.常规用户名称、密码验证</br>
2.用户登陆后可以编辑和删除个人留言，可以回复信息，但无删除其他人信息的权限。</br>
3.每个页面加上用户信息判断。防止URL登陆。</br>
